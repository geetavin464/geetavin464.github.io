* __Shortcuts__
  * `ctrl + o` - open file
  * `ctrl + shft + o` - open project
  * `ctrl + n` - New file
  * `ctrl + shft + n` - New project
  * `ctrl + alt + l` - Shift focus to solution explorer
  * `ctrl + shft + B` - Build solution
