* __Sorting__
  * Choosing the right sorting mechanism given the problem constraints
  * Optimal Sort - worst case run time nlogn

* __Insertion Sort__
  * Inefficient
  * Hence, used only for smaller data sets or almost sorted sets
  * Quadratic n^2 worst case run time
